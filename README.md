# Portal Investe Franca 🍊

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização completa, incluindo layouts responsivos, animações e efeitos de hover.
* **JavaScript (Vanilla)**: Implementação de comportamentos interativos como a navbar fixa, accordions, scroll suave e animações controladas pelo scroll.
* **Fontes (Google Fonts & Typekit)**: Utilização das famílias Montserrat para títulos e Open Sans para o corpo do texto.
* **FontAwesome**: Biblioteca de ícones vetoriais para interface.
* **Intersection Observer API**: Utilizada para disparar animações e contadores numéricos conforme os elementos entram na área de visualização do navegador.

## 🎨 Identidade Visual

### Cores Principais
O projeto utiliza uma paleta de cores moderna e contrastante para garantir legibilidade e destaque visual:
* **Laranja (#FF6B00)**: Utilizado para destaques, CTAs e elementos de marca.
* **Navy Escuro (#0d1a2e)**: Cor principal de fundo para a navbar e seções de destaque.
* **Navy Médio (#1a2a4a)**: Utilizado para variações de fundo e elementos secundários.
* **Branco (#ffffff)**: Cor principal para textos sobre fundos escuros e fundos de seções claras.
* **Cinza Claro (#f5f6fa)**: Utilizado em fundos suaves e elementos de interface neutros.

### Tipografia
A escolha tipográfica foca na clareza e num aspeto institucional contemporâneo:
* **Títulos (Headings)**: Utiliza a fonte **Montserrat** com pesos 800 e 900 para garantir uma hierarquia visual forte e impactante.
* **Corpo do Texto (Body)**: Utiliza a fonte **Open Sans** com pesos 400 e 600, focando na legibilidade em diferentes dispositivos.
